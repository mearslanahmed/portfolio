## Overview

A robust movie recommendation engine that goes beyond simple genre matching by analyzing deep content features like cast and keywords to deliver highly relevant suggestions.

## Problem

Basic recommendation systems often rely solely on genres, resulting in generic suggestions that miss the nuanced similarities between films.

## Solution

A content-based approach utilizing natural language processing and vector math to calculate true similarity scores across multidimensional movie data.

## Features & Architecture

- **Intelligent Engine**: Uses `CountVectorizer` and cosine similarity to factor in cast and keywords—not just genre—for much more accurate and relevant suggestions.
- **Serverless Architecture**: The entire system runs as a single Python file deployed on Streamlit Cloud, requiring absolutely no traditional backend or infrastructure.
- **Data Pipeline**: Leverages Pandas and NumPy for efficient data cleaning, normalization, and feature extraction from raw movie datasets.
